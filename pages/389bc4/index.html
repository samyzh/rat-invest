# 持有的逻辑

有很多朋友，对短线、中线、长线不理解，其实这个并不是按照时间来划分，

  • 短线是在热点、人气足的时候，题材投机，以小博大，设定好止损，做短期爆发。

  • 短线博弈，可以先用底仓试错，如果顺手连续赚钱的话，可以加大点仓位，因为你的逻辑和市场的逻辑是匹配的，稍微激进点没有问题。

  • 但如果第一次出现错误，就要回到最开始的底仓去博弈，进行试错，重新开始。纪律是短线最重要的品质，千万不能短线变中线，中线变长线，然后熬成股东。

  • 中线主要是跟随市场主线，趋势很重要，寻找预期差，要么干主线龙头，要么干估值和位置低有望补涨的品种。跟随是中线最重要的品质。

  • 长线主要在于价值，时间越长基本面的重要性越强，要有穿透未来的眼界，找到好的生意模式，未来空间大，成长性强的行业，选择好公司，等待好价格。耐心是长线最重要的品质。

# 催化剂

好公司 一般的套路 都能选出来 ；估值 太难了 ；选时 催化剂跟踪 太累了

  • 1)盈利大超预期；
  • 2)高管或大股东增持；
  • 3)跌不动时（坏消息出来股价不跌甚至上涨）；
  • 4)基本面拐点；
  • 5)新订单、新技术突破、新产品、新管理层；
  • 6)券商研究员出深度报告或提高评级；
  • 7)融资需求（定增、发H股）；
  • 8)股权激励；
  • 9)政策出台；
  • 10)高送配；
    1. 竞争对手因突发事件歇菜；
  • 12)海外同板块公司飙升；
  • 13)流通股东中机构投资者和超级大户终于出去了；
  • 14)项目达产；
  • 15)开拓新市场；
  • 16)参股公司上市；
  • 17)深藏不露非仙即圣，底部拉起持续放量。

# 进场前考虑因素

止盈离场也类似

我总结了一些心得，是亏钱亏出来的。

所有的交易心理障碍都来自于浮亏，大的浮亏有大的心理压力是避免不了的，所以最好是进场之后尽量不产生浮亏，要么不买，要么买对

进场前先定义交易的时间级别，分成两个，进场级别和持有级别，进场级别通常比持有级别要低一级，例如持有级别是周线，那么进场级别就是日线。

要分开的原因主要是来信息接收的滞后，例如当你发现一个行业股票的机会，产业趋势会持续一年以上，可能周线级别启动的最佳进场位置已经过了，所以只能找日级别机会进场。

# 进场前考虑因素

  1. 宏观环境是否对这笔交易在持仓期限内有利；
  2. 品种基本面是否有利；
  3. 技术形态在进场级别是否有利（做多是在支撑位附近，做空是在压力位附近）；
  4. 动能指标在进场级别是否有利（做多是由弱转强，做空是由强转弱）；

有利+1分，不利-1分，中性0分，2分或以上可以做，3分或以上要重仓，4分加杠杆上；

框架是无法直接转变成行为的。换个思路给大家参考，你能容易找到的投资大概率都是风险与收益匹配的。也就是说容易找到的只有低风险低收益，高风险高收益两种。剩下的低风险高收益，不容易找到，所以你会发现看不懂，不敢进场的机会多了去了。高风险低收益，傻子才做。

所有投(du)资(bo)技巧都是风险收益比的优化技巧，对于你找到的高风险高收益机会，你的优化目的就在于如何降低风险，能做的有技术分析择时、成长股分散持仓降低个股与行业的风险暴露、选队友抱主力集团军大腿。每年找新的抱团方向，突破后跟进就是这几个套路的综合运用。

对于低风险低收益的机会，你的优化目的就在于如何提高收益，能做的有技术分析择时、行业与财报的筛选等。在低估值高股息股价低波动的票里面做均值回归型买卖就是这几个套路的综合运用。

# 离场

经典价投卖出股票只有3个原因：

  • 一是基本面恶化；
  • 二是价值严重高估；
  • 三是出现了更好的投资机会；

基本面恶化 不计价格都要卖，这个没什么好纠结的；

严重高估 就是艺术了 感觉得用估值+技术分析来卖；

出现更好的机会就更离谱了 除非是一个骆驼VS马 肉眼可识别 不然感觉没必要；

# 业绩驱动

大佬们教的更适合10变100，欧奈尔教1变10，0怎么变1自己想办法

翻来覆去，发现还是业绩驱动的趋势投机行情最容易上车，基本都能找到集团军抱团，暴毙风险小。

纯价投现在市场不走这个风格，强行坚持别人都涨翻天，留着自己独守滞涨股空虚寂寞冷，心理关卡难跨越，目前搞过于反人性。

现在就是业绩+资金情绪驱动的行情。

而且最重要的驱动因素货币宽松如果消失甚至反转，按照现在情况发展那就是大熊市，价值股也照样被锤
