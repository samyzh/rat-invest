# 背景

# 门店

# 地点

  • 前期现在老家做，积累开店经验；期间多关注区里面很有潜力片区的不错的店铺情况；
  • 明年后期，考虑区里面开门店，做食品及fs门店多门面方案；

# 装修

  • 装修风格；参考文化新城fs相关店面装修；
  • 可以考虑，太极八卦元素；
  • 祖传fs的来历，及作过某些大型祠堂，等重大事件来推广自己；
  • 门口招牌优化；可以自己参考网上设计方案；
  • 门更新优化及店面吊顶；可以参考装修师傅方案；
  • 不错的书摆设，或者木匠相关工艺推荐；
  • 泡茶桌子，装修有上档次点，方便后面用于抖音视频推广宣传；

# 附属销售

  • fs相关附属；或者日常开销使用；
  • 或者快递点相关；

# 运营

总体情况：

  • 下乡情况下，多拍点视频，保存好方便后面推荐，用于题材处理；
  • ‌借助旁边银行办理点，做宣传，所以广告牌要靓，让旁边的两两门店以你的门店牌子为找目标 ‌运营，道教相关文化，及弗教相关，平安符，普，S族文化。
  • s族fs文化馆，z氏fs文化馆，可以卖香烟蜡烛，平安符;
  • 取名字业务;
  • ‌结合最新的新商品房业务;
  • ‌发表对小区的整体看法，

# 周边社交

  • 铺设好门店，泡茶及周边相关行业人员，多沟通，多串门；
  • 结合阿公相关，可以做相关的题材推广；

# 抖音【要点】

适合圈内群体社交；同步最重要的推荐；在乡下抖音是很好的传播途径；

视频题材一样要发布到同城区域；

# 内容运营点

  • 下乡实践考察视频，不错地点，及访问做推荐分享；先从开始玩，到后面多人推广做精品；
  • 传统文化，fs相关知识分享；
  • 结合国家传统节日，做视频题材宣传；
  • 不错题材视频，我们可以后期处理修改，借鉴其他相关fs号；
  • 结合抖音热门题材做抖音推广，后期还可以做相关的广告投放；
  • 在稳定的粉丝后，可以慢慢推荐抖音线上商城，做fs及木匠相关附属衍生商品；
  • 门店开业，N周年相关庆典，搞活动，拉人气；
  • 结合镇上相关网红，或者题材，畲乡做相关的配合题材；

# 微信

适合熟人社交；加好友，多分享；定时微信朋友圈传播，后面定期视频号运营；

利用上面抖音视频；

# 数字化

后期数字化，记录个人客户相关信息；方便节假日回访；【三响炮】打法；

稳定老客户，拉动新客户；
