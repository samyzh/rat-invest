# 简介

# 概念使用

# 资产软，负债硬，现金为王，摸鱼不成，默默搬砖

这些原理貌似可以用在新上市股票的炒作? 新上市的公司，财报通常经过装修，一时半会不会掉，故事越大越远，公司知名度越高，就越值得炒，根本不用管基本面，只需要关注故事是否短期不能证伪。反正在蜜月期，没有消息就是好消息；

没那么复杂。 有时候看线就行了。刷下财报评分，看看机构发文，逻辑顺畅加自选。突破了就上车 跌破20就卖。

# 总结方式【要】

进入财报助手后应该先关注评分，通过分数区间对公司的健康程度有个大概的了解，再通过标签了解公司的风险点在哪里，或者有哪些亮点值得关注的。接着再进行更细分的分析；
