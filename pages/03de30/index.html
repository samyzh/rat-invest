投资与创业

# 普通人积累财富最靠谱的3个途径

# 1、一份高性价比的工作

在我看来，工作有两大意义。

一是提供稳定的现金流，这一点大家应该有亲身体会，不做过多的解释。

二是缓解人生的无聊与寂寞。马克思说，人的本质，是其所有社会关系的总和，这个观点我非常认同。一个人如果长期离群索居，往往会陷入空虚与无聊，特别是自控力不佳的人，工作可以提供一个不错的社交网络，工作的内容也可以让你的生活更为节律。我见过很多已经实现财务自由，但仍在极其勤奋工作的人;

最好的工作，类似于可转债，向下有工资来做保底，向上有期权、股权等高弹性的回报。每年的春天，都是跳槽的高峰期，祝大家做好各自的人生选择。

通过做好本职工作，来积累财富，这是绝大多数人最靠谱的选择。

# 2、一份自己拥有绝对控制权、现金流良好的实体生意

很多人都有创业的冲动，但创业是一件成功率很低的事情，结合我的亲身经历与所见所闻，分享几点心得：

（1）如果有做同样生意的上市公司，除非你非常自信可以做得更好，否则还是买上市公司的股票最省心。比如说，与其现在创建一家空调公司，不如把这钱直接去买格力的股票。上市公司一般都是相对成熟的公司，具备了一定的规模优势，如果有自己看好的生意，并且公司团队值得信赖，不如直接去投资他们。当然，那些为梦想而创业的朋友除外。

（2）一定要自己掌握控制权，虽然未必亲自打理。只有当你真正有控制权的时候，这家公司才真正是属于你的，你才能根据你的实际需要，调配资源。

（3）现金流一定要非常非常好。做任何生意，现金流都是生命线，一个生意，哪怕概念再诱人，如果不能持续产生现金流，那么就很难长久维持经营。

# 3、证券投资

通过工作和实业，你会积累大量的资金，有了本金，自然需要做投资。

最适合普通人的投资方式，就是投资指数基金；如果想有超额收益，就需要用心选股

联系

实际上，工作、实业与投资，这三者密不可分。

工作和实体生意，会增进你对投资的理解，为你的投资提供源源不断的现金流，在你持仓回撤时，因为有工作和实体生意的支持，你不会有太多的心理压力，而且有新增的子弹去补仓。此外，工作与实体生意，还有助于你保持良好的投资心态，不急于赚快钱。

投资和实体生意，会让你在工作中，更有底气与尊严，不需要违心去做一些自己不喜欢做的事情。

投资这件事情，最能锻炼一个人的长远眼光，这有助于你在工作和实体生意中，进行一些关键的抉择。巴菲特曾说，我是一个好的投资家，因为我是一个好的企业家；我是一个好的企业家，因为我是一个好的投资家。也正是这个道理。

这三个途径中，有的人只适合一个，比如说工作，有的人只适合两个，比如说工作与投资相结合，有的人这三项都很擅长，但无论是选择哪一条道路，只要适合自己，并做到极致，都会积累很多财富。最理想的境界就是，做自己喜欢做的事情，然后顺便赚到钱。

# 投资与创业的比较

如果创业，那么，一定要自己掌握控制权，虽然未必亲自打理。

只有当你真正有控制权的时候，这家公司才真正是属于你的，你才能根据你的实际需要，调配资源。

投资，是普通人性价比最高的创业。

做投资，不需要你做大额的成本投入，不需要你打点各种关系，不需要你做大量的体力劳动，即使投资失败，最多亏光本金，也不会让你负债累累。

创业需要从零开始，但投资却可以让你有机会，选择全世界最优秀的公司、最优秀的团队、最优质的资产。即使你现在去创业，也不可能打造出第二个腾讯，但你可以直接买到腾讯的股票。

最可贵的是，投资是一个与时间做朋友的行业，年岁越久，你的经验也就越丰富，和别人竞争时，你也就越有优势。

让天才们去创业，我们这些普通人，只需要在他们将来成功上市后，安心分一杯羹，就足够了。

我个人认为，你最好先认真考虑一下，自己的优势到底是什么，是在创业，还是在职场，然后再做最终的决定。

每个人都有自己的相对优势，用错地方，不仅是资源浪费，还会引发一连串的恶果。

无论早年工作时，还是现在做投资，我自己的一贯原则都是，只做自己有相对优势的事情。

我相信，以你的能力和悟性，肯定能找到自己的相对优势，并把它发挥到极致。

当然，对于绝大多数人来说，最合适的选择还是安心上班，同时做好投资，双轮驱动。

必须要说明的是，我建议大多数朋友「安心上班，同时做好投资，双轮驱动」，并不是建议大家直接去买股票，这完全是两回事。

股票只是投资工具中的一种，股票投资对专业性的要求极高，并不适合大多数普通投资者。

# 期权 (opens new window)

拼多多期权的归属加解禁期长达7年，可能也和公司自身上市速度较快有关，延长解禁期是为了保持员工稳定性。百度上市用了5年，腾讯6年，阿里巴巴15年，京东16年，拼多多则只用了3年。

部分公司会采用第1-2年没有期权，后面再逐年发放的规则。以上人力负责人表示，公司会将前1-2年设置为考察期，如果不满意员工业绩，就在期权开始归属前开除员工，就避免授予不适合公司的员工期权。而员工往往会为了期权接受降薪，就相当于降薪工作了1-2年。

闯过以上两关，员工来到了行权阶段，这也是矛盾的集中爆发期，因为终于到了公司和员工摊开算账的时候。

如果公司不回购员工的期权，那员工就可以把归属后的期权带走，相当于持有公司股份，等公司上市后再变现。

一位处理过多起期权纠纷案的律师表示，公司回购期权的价格，一般是以公司上一轮融资完成后的估值为基准，打个折扣定价。冯大辉认为，按照公司2014年完成C轮融资之后的估值，每股价格已经超过30美元，双方商定回购的价格是18美元/股，如果公司不想回购，自己也接受全部行权。

# 投资，是性价比最高的创业

餐饮行业，几乎没有任何门槛，这样的行业里，竞争是非常非常非常激烈的，残酷的竞争下，很难留存下多少利润。

中美股市，这么多年、这么多家上市公司，餐饮行业的长牛股，就只有麦当劳和肯德基（百胜餐饮）。

行业分三种：辛苦不赚钱，赚辛苦钱，躺着赚钱，很不幸，绝大多数餐饮企业都是第一种，极少数的顶尖餐饮公司，做到了第二种，比如说海底捞。

买商铺，在房价不大涨的情况下，至少需要十年以上才能收回成本，而我们做投资，无论是买股票、买基金，基本都是可以随时变现的。

职场很艰难，所以，总有人想搞搞副业，或者直接出来创业。

我想说，投资，是普通人性价比最高的创业。

做投资，不需要你做大额的成本投入，不需要你打点各种关系，不需要你做大量的体力劳动，即使投资失败，最多亏光本金，也不会让你负债累累。
