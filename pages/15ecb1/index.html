基础名称相关

# 绝对/相对收益

绝对收益（Absolute Return）和相对收益（Relative Return）

基金的收益目标，可以分为绝对收益和相对收益。

  • 相对收益，就是说战胜大盘的收益。比如大盘涨了10%，基金能做到涨12%，或者大盘跌20%，基金只跌20%。所以这类基金追求的是战胜大盘。
  • 绝对收益，就是追求绝对的正回报，不论市场是上涨还是下跌。比如大盘涨了10%，基金能做到涨8%，大盘跌10%，基金能做到转2%，那么这个基金始终在赚钱，这样通过多年的复利就能放大收益。

相对收益相关的产品，大部分都属于公募。

**绝对收益相关的产品，大部分属于私募。**绝对收益产品在国外是越来越少了，因为难度比相对收益要难不少，但是绝对收益却被富豪们喜欢的，因为富豪们已经很有钱了，比较求稳，如果一个基金产品是绝对收益，并且夏普率很好，那么一定是疯抢的。

# 场外/场内基金

从交易的方式，可以分为场内基金和场外基金。

场内基金，就是在市场内交易的基金，用户需要通过证券公司的交易软件买卖，这种类型的基金和股票类似，在交易时段，投资者根据买方报价和卖方报价，自己决定交易时机。这里的场内是指二级市场。

场外基金，投资者只能通过银行柜台、证券公司、基金公司的柜台、网站和移动应用等渠道来申购和赎回，这种基金每天也只有一个净值报价。

# 申购和赎回

这两个都是场外操作：

  • 申购，就是用钱申请购买基金份额。
  • 赎回，就是把基金份额退还给基金，取回钱。

说白了，你在场内交易股票、基金，都是和其他交易员做交易。在场外申购和赎回基金，是和基金公司做交易。

申购和赎回都要交手续费，申购以后会有一段的封闭期，短的几天，长的几年，如果要提前赎回，一般需要交额外的手续费。

# 区别

主要区别：

  • 交易方式不同
    • 场内基金：和股票交易一样，你是在通过券商来和其他投资者交易。
    • 场外基金：你是在向基金公司申购和赎回，无论你是通过什么渠道。
  • 管理费率不同
    • 场内基金：费率相对低一些，买卖就和股票一样交同样的佣金，也没有申购费和赎回费。
    • 场外基金：费率相对高一些，同时申购和赎回都有费率，一般申购以后会有一个封闭期，如果提前赎回会有额外的费用。
  • 流动性有区别
    • 场内基金：由于可以在市值中交易，所以买卖人数活跃，流动性较好，价格变动频繁，可能产生溢价和折价。
    • 场外基金：每日公布一个净值。
  • 适用人群
    • 场内基金：更适合愿意花点时间花精力学习炒股、享受交易乐趣。
    • 场外基金：更适合不愿意花时间花精力在炒股上，只想赚取一分收益的人。

# 单位/累计净值

# 单位净值

也简称基金净值，是当前的基金总净资产除以基金总份额。假设一个基金成立以后发行，募资100万，初始份额分成了1万份。基金的初始净值是100万/1万，等于100，此时假设你花500元去申购，你会拿到5个份额。

然后基金拿着募集到的100万去投资，1年以后，赚到了20万，此时份额不变的情况下，净值就变成了120万/1万=120，而你的500元也变成了5*120=600元，此时再想要申购1个份额，就需要花120元了。

当然，这些是简化的情况，实际情况是基金不一定立刻就能募集到100万资金，另外基金所购买的资产价格也在变化，而且每天都会有人申购、有人赎回，而申购和赎回不仅会影响份额，同时都需要交费，这些费用也会变成基金总净资产的一部分，所以基金的总净资产在不停地变化，份额也在不停地变化，基金的单位净值计算就非常复杂，基金公司都有专业人来计算。

基金公司一般每天至少会公布一次单位净值，这样用户才能根据净值决定是否申购和赎回。不过有的基金，比如封闭式基金可能会每周或更长时间才公布一次。

值得一提的是，有时候大规模的赎回也会导致基金净值变化较大，因为赎回费会计入到基金的总净资产中。

# 累计净值

累计净值。这里还是用前面那个例子，基金募集到初始资金100万去投资，初始净值是100，1年后赚了20万，此时基金总净资产变成了120万，假设没有分红和拆分股份，那么此时单位净值是120万/1万=120，累计净值此时也是120。

现在基金觉得每份分红10元，则分红后，基金总净资产变成了110万，单位净值变成了110万/1万=110，但是，累计净值没有变化，还是120，怎么算的呢？是单位净值+分红。

此时，基金又决定拆分份额，1拆2，则1万份额变成了2万份额，那么单位净值变成了110万/2万=55。那么，累计净值有变化吗？答案是没有，此时，累计净值的公式是： T日基金份额累计净值=(T日基金份额净值+拆分后份额分红金额)*拆分比例+拆分前分红金额

所以就是(55+0)*2+10=120，换句话说，累计净值，不论是分红、还是拆分份额，都不会影响其值，它是真反映基金在运作期间的历史表现的指标，投资者在评估基金表现时，应该关注的是基金份额累计净值。

# 基准指数

（benchmark index）

评估基金的表现，需要拿一个参考对象，一般来说，会拿一个指数作为比较，也就叫做基准指数。不同的基金对比的基准指数不同。

比如你成立一个美股的科技股基金，但是美股市场已经有纳斯达克100指数基金这种回报可观的科技股ETF基金了，别人为什么投你呢？所以你需要证明自己别人才会买的基金产品，那么你可能需要使用纳斯达克100指数来作为基准指数对比。一般一个基金发行时，在官网介绍页上会提到基准指数，或者默认是和大盘指数对比。

# 共同/对冲基金

共同基金（Manual Fund）和对冲基金（Hedge Fund）

美国的场外基金，也可以分为共同基金和对冲基金。

  • 共同基金，Mutual Fund。
  • 对冲基金，Hedge Fund。

两者本质上都是基金，从一群投资人手中募集资金，然后基金管理人用这些资金去投资，所有投资人共享赚取的收益和承担的风险。

区别是：

  • 投资门槛不同
    • 共同基金：资金门槛较低，但普通人也可以申购。不过共同基金也有一些大资金客户，比如国家级的养老金等。
    • 对冲基金：资金门槛较高，一般至少几十万刀起步，也有净资产要求。另外，对冲基金一般来说，有申购人数和资金限制，当基金的投资人超过一定人数，或者募集到的资金超过一定数量，基金可以不允许再申购。
  • 投资方式不同
    • 共同基金：一般有严格的要求，因为养老金这些机构，可不希望资金拿去投机、做空之类的。所以一般共同基金的投资方式都比较简单，透明，而且普遍以长线为主。
    • 对冲基金：投资方式较多样化，比如可以做空、可以使用期权、可以使用掉期、可以上高杠杆等。

简单的说，对冲基金是富人玩的游戏。对冲基金相对共同基金，是从富人手中募资，收的管理费较高、员工薪资也较高，当然，竞争也比较激烈。富人们投了钱，如果几年不能盈利，必然会撤资，所以对冲基金的压力也很大，寿命短的对冲基金并不少见，而那些长期存活下来的对冲基金，都有自己独道的交易理念和策略。

在世界顶级对冲基金中工作的人，无论是能力、还是薪资，都是一流的。

# 公募/私募基金

这两个是中国的概念，其实和美国的共同基金、对冲基金非常类似，所以也可以简单的认为

  • 公墓基金，即中国的共同基金。
  • 私募基金，即中国的对冲基金。

# 开放/封闭式基金

# 封闭式基金

封闭式基金，Close End Funds，简称cef，它有几个特点：

  • 申购以后，不可赎回，必须持有到期。有些封闭式基金会上市，所以可以在场内交易。
  • 基金一旦达成募资目标金额，或者截止时间，即会处于关闭状态，不可再申购。 虽然特殊情况下此类基金可进行扩募，但扩募应具备严格的法定条件。

我们知道场外基金申购以后，会有至少几天的封闭期，你不能第1天申购、第2天就赎回，如果大家都这么玩，基金就没法拿钱去投资了。封闭式基金的优势是，不需要面对用户赎回的问题，可以放心大胆的专注中长线投资，坏处是用户在封闭期没法赎回钱。

总体来说，封闭式基金在逐渐没落，被开放式基金和ETF取代。盘子也相对开放式基金和较小。

# 开放式基金

与封闭式基金对应的是开放式基金，这类基金就是最标准的，只能申购和赎回的场外基金，不能在场内交易。

共同基金一般为了应付有人赎回，投资的仓位不会用到100%。尤其是有时候大规模赎回，对净值影响比较大。

# LOF基金

上市型开放式基金（LOF）

前面提到一般开放式基金只能在场外申购和赎回，不能在场内交易。中国创新发明了一种上市型开放式基金，英文名Listed Open-Ended Fund，简称LOF，算是开放式基金的改良，既可以申购和赎回，也可以在交易所交易。

# ETF基金

ETF基金，这个是目前最流行的基金种类，英文名Exchange Traded Fund。顾名思义，这种基金天生就是在交易所交易的基金，场内基金大部分是ETF基金，尤其是美股市场上，有2000只左右的ETF。

ETF基金如果是指数基金，跟踪的是一揽子股票的话，那么可以在场外申购和赎回的，不过这里申购和赎回不能用钱，而是必须用一揽子股票，比如跟踪180只股票的上证180 ETF，

  • 申购，需要购买100只成功股的股票，去换取份额。
  • 赎回，可以拿ETF份额，换回成分股股票。

# ETF和LOF比较

ETF和LOF都可以在场内交易，也都可以在场外申购和赎回，主要区别是什么呢？

  • ETF申购和赎回都是用股票，所以是在场内进行，必须有证券账户，而LOF申购和赎回都是用限价，既可以在场内也可以在场外，在场外只需要有基金账户。
  • ETF并不需要准备现金应对赎回，无论是在场内交易、还是场外申购赎回都一样，在场外赎回就相当于是在和基金经理做交易。而LOF需要准备资金应对赎回。
  • ETF没有现金转换步骤，所以跟踪误差要比LOF小。
  • ETF的申购和赎回门槛高很多，因为要手工按比例购买成分股，普通人很难做到，一般都是机构的程序去实现，并且是有资金要求。而LOF普通人都可以申购赎回。
  • ETF一般管理费要比LOF低。

# 货币基金

货币基金，英文名Money Market Fund，简称mmf，是一种开放式基金，是聚集社会闲散资金，由基金管理人运作，基金托管人保管资金的，专门投向风险小的货币市场工具。

货币基金的特点：

  • 申购资金门槛极低，甚至1元都可以申购。
  • 风险极低，甚至可以说是最低风险的基金。因为货币基金投资的包括短期国债、银行票据、银行背书的商业汇票、银行承兑汇票、银行定期和大额可转让存单等，期限通常限制在一年内。
  • 手续费低，很多货币基金没有申购费和赎回费。
  • 货币基金赎回非常快，约几天。

货币基金产品主要有有两个指标：

  • 每万份收益，顾名思义，就是每一万份份额的货币基金在过去的一天产生了多少收益。
  • 七日年化收益率简称“七日年化”，以百分比呈现。比如说“七日年化”是2.273%，它的意思是如果今后一年都维持过去7天平均每天的收益率不变，投资者持有一年后能有2.273%的整体收益。需要注意的是“七日年化”代表的是过去7天赚钱的速度，未来的收益可能会下降、也有可能保持不变、也有可能会上升。

货币基金虽然自己门槛低，但是很多人持有少量闲钱，不想买高风险的股票或基金，就会购买货币基金，所以货币基金聚少程度，总体规模也很大。大部分基金公司和券商的网站和移动应用都会销售货币基金，很多理财产品后面对接的也是货币基金，比如中国的余额宝。

# FOF基金

也就是基金中的基金，Fund of Funds，简称FOF。与普通基金的区别在于，普通基金投资的是股票、债券等证券，而FOF投资的是其他基金，它通过对不同板块和主题的基金进行筛选来投资，进一步将投资分散化，抗风险能力更佳。
