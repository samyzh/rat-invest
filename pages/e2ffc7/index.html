# 中国联通 (opens new window)

研究了一个晚上的联通，感觉情况比想象中要好很多，非常象N年前的东航:

  • 1.关于5G投资:大规模5G建设要大概三年，100万基站，就能实现关键区域的覆盖，也就是说从2023年开始，资本性支出会好很多，不过在比期间，公司的分红会保持稳定，这个应该也是这段时间下跌的原因。不过2024年应该是可以看看到分红上升了，毕竟自由现金流实在太好了！如果这个没错，感觉2022年-2023年可能有大行情
  • 2.关于客户数量下滑:公司现在更重视用户的质量，主动放弃低价值用户，导致APRU上升，营销费用下降。另外公司预计今年2G全部退网，3G可能还要一定的时间，未来实现5G关键区域覆盖，4G广覆盖，两层的网络结构（另外王总对于700mhz共享的信心很大）由于2g的退网，会部分覆盖5G成本的上升。
  • 3.关于利润平移:与电信一样，公司都有一些会计处理，单年计提了50亿的应收减值，多提了25亿，公司表示有一半是可能收回的，就是不知道2G3G退网，会不会增加报废。
  • 4.关于混改:未来云南广西的模式应该会调整，公司现在的资金情况，远远好于当年混改时了，未来会有足够的资金自己投资了。公司现在的情况确实是大幅度好转了，账上净现金初估了一下，应该有200-300亿了，确实也不在需要象以前一样了。
  • 5.关于资本运作:公司资金非常好，不需要再融资了，但特别提到会考虑下属公司的ipo，应该是有动作的了！首先应该是招联，十多亿的利润！
  • 6.关于资产:资产负债表很干净，200-300亿的净现金，400亿港币的铁塔，扣除以后只有不到800亿的市值了。每年的自由现金流可是有300-400亿啊！
  • 7.关于未来收益:公司预计EBITDA率会保持稳定，收入增长应该会保持稳定，因此只要今年报废什么的不要调节，2021年的净利润应该是会有比较好的增长！

# 不错的文章记录

“无论你是做实业，打工族，还是做投资，你一定要记得财富升级定律：钱真的不是一点一点赚到手的，而是某一天你开窍了，在能力具备的情况下，偶然抓到了某个机遇，从而一下子赚到的！

所谓开窍了，就是你有了这个认知；所谓能力具备，需要你的长期积累和眼光；所谓某个机遇，需要你耐心去等待和观察；所谓抓到，就是该勇敢赌一把的时候绝不畏畏缩缩；所谓一下子赚到，就是你的财富是非线性增长。”

“对于想要追求财富升级的投资者来说，真正的复利是机会复利。假设我开始有10万，抓到了某个机会，一下子变成百万级别了，对于我来说，这个就是识别机会并把握住机会一下子把钱转到了。接下来，后面没什么机会，我拿一点点资金打打酱油，找找感觉，后来又发现到一个机会了，我把上次赚到的百万再继续进行投资，又抓到了一个机会，从百万变成千万甚至亿级别的，从6位数的人变成了8位甚至9位数圈子里的人。”

“我是学经济学和金融学出身的，但我并不相信我所学的各种经济金融理论，经济学本身是漏洞百出的科学，然后编造了各种新的理论去补那个漏洞，就像是说了一个慌之后，又编了好几个谎言来掩盖原来的谎言，以致于正方和反方都能够获得诺贝尔奖；金融学中创造的各种金融工具，本质上只有一个目的，那就是割韭菜。

整个金融市场割韭菜的基本套路都是两步走：

第一，让投资者把手中的钱去变成某种资产，然后让这种资产升值，吸引更多钱进来；

第二，资产升值之后，想方设法赚取流动性离场。这种资产可以是股票、商品、数字货币、房产等等。”

“因此，金融市场的本质就是资金与筹码的交换，低位想方设法骗取你的筹码，高位想方设法骗取你去接筹码。什么股票定价模型，PB，PE的，都是术，不是道；什么价格发现，套期保值，都是术，不是道；什么区块链，分布式，都是术，不是道。

当然，必要的时候，资本经常会采取一招：make new news。这就像是像水里扔一块石头，顿时激起千层浪，资产价格大幅波动，吸引无数资金冲杀进来，然后资本来实现他们想要的资金与筹码的交换。”

这也是最近这轮我选的主要是去年盘整，今年才启动和创新高的票的原因，机构手上筹码不多，会让行情有更大的空间；

# 黄金

山东黄金，黄金的理由已经在前文《网页链接》通胀阶段涨工业金属，滞涨阶段黄金上涨，黄金具有后周期属性，复盘71-79年，03-11年，都可以看出等到工业金属上涨完，才是黄金的主升阶段。今年黄金完成探底，黄金股也完成探底，进入了黄金反弹，而黄金股还没有来得及上涨的阶段。黄金股下跌空间较小，而上涨潜力巨大。紫金矿业同时生产黄金和铜，由于铜能抗通胀，金有抗滞涨，两种产品有具有自我对冲作用。

中国船舶，经过近十年的行业萧条，造船行业完成去产能，民营造船所剩无几，造船行业龙头集中度大为提高。股价经过10年的下跌，也探明底部在14元左右。经过海事和造船行业的雪友的电话咨询，造船行业明显进入行业景气阶段，只是从行业景气到释放利润还需要一段时间。
