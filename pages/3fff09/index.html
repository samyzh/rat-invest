散户高位逃顶，低位抄底的妙招

# 均线理论

• 均线的支撑作用

• 均线的压力作用

• 5日均线，10日均线—短期

• 30日均线，60日均线，120日均线—长期

# 克罗均线策略

# 买入信号

1）长期买入信号：收盘价>MA10>MA20>MA50 （MA指移动平均线）

2）短期买入信号：收盘价>MA4>MA9>MA18

以上两个信号出现一个即可做多。

# 卖出信号

1）长期卖出信号：收盘价<MA10<MA20<MA50

2）短期卖出信号：收盘价<MA4<MA9<MA18

以上两个信号出现一个即可做空。

# 散户为何容易卖飞和深套

  • 面对赢利——风险厌恶（落袋为安）
  • 面对损失——风险偏好（赌一把反弹）

结果：

  • 牛市——卖飞
  • 熊市——深套 赢

# 散户的心理

宁愿被套，不想踏空

踏空的痛苦值：100

套牢的痛苦值：90

赚钱的快乐值：80

散户亏钱的根源：

1.学习能力差

2.无法逆人性

# 止盈方式

1.静态止盈 ——设立具体盈利目标位

2.动态止盈 ——价格回落幅度、均线破位止盈、技术 形态止盈 赢

# 我的动态止盈方式：

1.预期不会继续上涨、风险降至（猪肉，鸿蒙）

2.情绪达到顶点、股价出现巨大泡沫（2600元的茅台， 证券，银行）

3.趋势短做、分批止盈（有了利润，先减仓，心态可稳 中远海控，东方银星，赣锋锂业）

# 止损方法

1.静态止损 ——设立具体的止损目标（定额、保本）

2.动态止损 ——技术止损、市场发生系统性风险

# 我的动态止损方式：

1.坚决避免崩溃出现，跌到成本必离场

2.高位天量大阴线、顶部反转形态、跌破趋势（东方通 信，君正集团）

3.系统性风险、个股逻辑发生变化（股灾，猪肉，汇顶 科技）

# 顶部反转形态

# 卖出的具体要领

1.重要均线，趋势线跌破（海马汽车）

2.量价关系，股价上涨，成交量缩量—资金锁仓（百 润股份） 高位放量—资金出逃

3.龙虎榜— 找到机构出逃，外资出逃，及时躲避

4.看情绪，缩地量能否走反击，该强不强则为弱 该弱不弱则为强

# 比较确定性的买入技巧

• 追随大阳线买入法（启动点阳线必须超过7%）

• 出强度（过去三个月，有涨停过，量能配合良好）

• 不要有连板（往往趋势牛股走2连板以上是很容易 短期见顶的）

• 是趋势型上攻走势（以上三点符合过后，还有很多 种牛股走势也很牛，但不符合趋势走法）

# 趋势的攻击形态买点

• 大阳线—连续上攻的K线后出现涨幅7%以上的 大阳线，过去有强度，趋势上攻形态有弧度，不 要脉冲式阳线（复星医药，妙可蓝多）

• 反包阳线—重塑趋势，不破坏原有趋势（王府井）

• 趋势首板—趋势牛股也会有涨停板，趋势股的首 板也存在惯性的上冲动力（人民网）

# 实战注意事项 【要点】

• 没有强度的个股要小心，很容易半路夭折

• 走势越漂亮越好

• 止损要严控，一般在10%左右，跌破趋势止损

• 观察期在1-3天，取决于市场环境

# 实战所需要素

• 题材要热门

• 人气要高

• 有量能支撑

• 形态优美

• 极品趋势牛不多，但是出现就是大牛
