常用股票术语

# 名称说明

# 换手率

（Turnover Rate）

换手率，指在一定时间内，一只股票的成交量除以该股票流通股的总数，这也是一个反映股票活跃度的指标。换手率越高，说明越活越，但换手率太高，说明投机分子很多。

比如某国一天的成交量10000股，流通股才8000股，所以换手率是125%，那么假设一个人A持有全部的8000股，就算他全部卖给另外一个人B，最多也才8000股的成交量，现在多出来的2000股成交量怎么来的呢？那就是B买了8000股以后又立刻卖出了2000股给人接盘，所以换手率太高的话，说明愿意持有股票的人很少，买了立刻就卖的人很多，所以买卖这个股的交易者投机性很强，股价有暴跌的可能，要小心。

# 看涨、看跌

看涨，指投资者看好某个股票的价格会上涨。 看跌，投资者看好某个股票的价格会下跌。

# 头寸、仓位、筹码

头寸：也叫仓位，指持有的某只股票的数量。 由于股票仓位很类似赌场里面的筹码，所以一般也称为“筹码”。

# 浮筹

浮筹：指散户手中的股票，流动性强。这部分股票持有者短线投机心态较强，操作时每拉升到一个关键的技术位，他们手中的股票就急着变现。

# 多头、空头

多头：因为看涨某个股票，而新建立一个做多的头寸的人，统称为该股票多头。 空头：因为看跌某个股票，而新建立一个做空的头寸的人，统称为该股票空头。

# 开仓、平仓

开仓：是指无论是做多还是做空，投资者在新建一个头寸的行为。 平仓：开仓之后，通过卖出或买入，来将仓位清空，换成现金，这个行为叫做平仓。以股票为例，新买入一只股票，即开仓，之后卖出这只股票，即平仓。

开仓和平仓在期货中用的比较多，做多也叫买开仓，做空也叫卖开仓，平掉做多仓位，叫卖平仓，平掉做空仓位，叫买开平仓。

# 利多、利空

利多：又叫利好，是指刺激股价上涨的信息，如上市公司经营业绩好转、银行利率降低以及其他政治、经济、军事、外交等方面对股价上涨有利的信息。 利空：是指对于空头有利，致使股价下跌的各种因素和消息，如上市公司经营业绩恶化、银行利率调高、经济衰退、通货膨胀、天灾人祸及其他政治、经济、军事、外交等方面促使股价下跌的不利消息称为利空。

# 长空、长多、短空、短多、死空、死多

长空：指是对股价非常不看好，长期做空的行为。 长多：指是对股价非常看好，长期做多的行为。 短空：指对股价短期不看好，短期做空的行为。 短多：指对股价短期看好，短期做空的行为。 死空：指非常不看好某只股票，卖出股票后，即使股价一只上跌，宁愿被套死也绝不平仓。 死多：指非常看好某只股票，买进股票后，即使股价一直下跌，宁愿被套死也绝不平仓。

# 翻多、翻空

翻多：指原本做空的投资者改变了想法，开始看好股票，不但将做空的股票回补回来，还买进股票。 翻空：指原本做多的投资者改变了想法，开始看好股票，不但将买入的股票卖出，还做空股票。

# 庄家、坐庄

庄家：指资金量大，足以影响一只股票走势的投资者，有可能是机构，也可能是大资金的散户。一只股票，能大涨大跌，一般都是要有庄家这种背后推动股票的人，所以他们是市场中的主力。 坐庄：指以庄家操纵某只股票，使股价走出趋势的行为。

# 散户、大户、中户、主力、机构

散户：就是买卖股票数量很少的小额投资者。 大户：指那些资金实力雄厚，投资额巨大，交易量惊人，能左右行情，控制市况的投资者，但他们同样是散户。 中户：指投资额较大的投资人。

主力：是持股数较多的机构或者大户。每只股票都存在主力，但是不一定都是庄家，庄家可以操控一只股票的价格，而主力只能短期影响股价的波动。 机构：指符合国家有关政策法规的投资机构等。

# 踏空

踏空：指投资者原本计划买入某股票，但一犹豫没有买入，之后股价上涨，投资者感到后悔的现象。

# 被套、站岗

被套：指投资者买入一只股票后，股价大跌，但投资者不愿意卖出股票，只能一直持有较长时间，等待股价涨回去。 由于这种行为，使得在等待期间资金没法利用起来，只能站在一旁看着其他交易员交易，就像被罚站岗的人一样，所以也俗称站岗。

# 诱多、诱空、多头陷阱、空头陷阱

诱多：指庄家有意制造股价上涨的假象，诱使投资者买入，结果之后庄家们迅速卖出股票，股价不涨反跌，让跟进做多的投资者被套牢的一种市场行为。 诱空：指庄家有意制造股价下跌的假象，诱使投资者卖出，结果之后庄家们迅速买入股票。股价不跌反涨，让之前卖出的投资者错过上涨行情的一种市场行为。 多头陷阱；指为多头设置的陷阱，通常发生在指数或股价屡创新高，随后迅速滑跌破以前的支撑位，结果使在高位买进的投资者严重被套。 空头陷阱：指为空头设置的陷阱，通常出现在指数或股价从高位区以高成交量跌至一个新的地点，并造成向下突破的假象，恐慌性抛盘涌出后迅速回升至原先的密集成交区，并向上突破原压力线，使在低点卖出者踏空。

# 跳水

跳水：指股价迅速下滑，幅度很大。

# 跳空

跳空：指股价受利多或利空消息影响后，出现较大幅度上下跳动的现象。也有上跳空和下跳空的说法。

# 阴线、阳线

阴线：股票K线图中，当日股价下跌的线叫阴线，也可以是当周、当季、当月。 阳线：股票K线图中，当日股价上涨的线叫阴线，也可以是当周、当季、当月。

# 阴跌

阴跌：指股价进一步退两步，缓慢下滑，如阴雨连绵，长期缓慢下跌不止。

# 阻力、支撑、盘整、突破

阻力：指股票上涨时始终无法涨到某个价格之上，每次接近该价格时就会遇到阻碍的现象。 支撑：指股票下跌时始终无法跌到某个价格之下，每次接近该价格时就会遇到支撑的现象。 盘整：指股价经过一段时间快速上涨或下跌后，遭遇阻力或支撑而呈小幅度涨跌变动的情况。 突破：指股价经过一段时间盘整后，突破阻力或支撑线的行为。

# 调整、反弹

调整：也叫回档，是指股价上升过程中，因上涨过快而暂时回跌的现象。 反弹：指在下跌的行情中，股价受到支撑暂时回升的现象，反弹后恢复下跌趋势。

# 高开、低开、低开高走、高开低走

高开：股票的当日开盘价高于前一交易日收盘价的情况。 低开：股票的当日开盘价低于前一交易日收盘价的情况称为低开。

# V型翻转

V型翻转：指股价一路下跌，然后经过一段时间，又一路上涨，价格形成一个“V”型的走势。

# 探底

探底：“底”就是大盘或个股定制下跌的一段区间。股价下行后，有大量的承接盘使股价回升，这个下行过程就是探底，目的在于探明该股的底部哪个价位有大承接盘。探底不成功后，继续下探，直到在某个底部价位出现大量承接盘使其难再下行时，称为探明底部。底部探明后，股价就要反弹回升。

# 震仓

震仓：指庄家在拉升的过程中使股价震荡，清洗筹码的过程，目的是尽量把心态不坚定的短线跟风盘甩掉。

# 追涨、杀跌

追涨：就是当股票上涨时，不管股价是多少都追着买入股票的行为。 杀跌：就是在股市下跌时，不管当初买入的股票价格是多少，都立刻卖出，以求避免更大的损失的行为。

# 止盈、止损

止盈：指当买入或做空股票盈利以后，为了锁定利润，提前平仓的行为。 止损：指当买入或者做空股票亏损以后，为了锁定亏损，避免进一步亏损，提前平仓的行为。

# 抄底、逃顶

抄底：在股票价格下跌的过程中，估计快要到底部，价格要从跌转为升的时候，就果断买入，这就是所谓的抄底。抄底能否成功，在于对“底”的判断是否正确，如果价格继续下跌，就是抄错了。 逃顶：在股票价格上涨的过程中，估计快要到顶部，价格要从涨转为跌的时候，就果断卖出，这就是所谓的逃顶。

# 补仓

补仓：指买入股票，股价下跌后，继续买入以将平均价拉低的行为，一般只有非常看好一只股票才会补仓。

# 对敲、打压、洗盘、吸筹、出货

对敲：是指庄家或大的机构投资者为了达到操纵股价的目的，蓄意串通，按照事先约定的方式或价格进行交易或互为买卖的行为。 打压：是指庄家用非常规的方法，将股价大幅压低，目的是会为后面大量买进。 洗盘：庄家为达炒作目的，必须于途中让低价买进，意志不坚的散户抛出股票，以减轻上档压力，同时让持股者的平均价位升高，以利于施行做庄的手段，达到牟取暴利的目的。洗盘动作可以出现在庄家任何一个区域内，基本目的无非是为了清理市场多余的浮动筹码，抬高市场整体持仓成本。 吸筹：是指在股市中庄家、主力或大户介入某一只股时，不断买入股票（建仓或加仓）的行为。 出货：通常指庄家在高价时，不动声色地卖出股票。

# 护盘

护盘：股市低迷、人气不足时，机构投资大户大量购进股票，防止股票继续下滑的行为。

# 坐轿、抬轿

坐轿：指预测股价将涨，抢在众人前以低价先行买进，待众多散户跟进、股价节节升高后，卖出获利的现象。 抬轿：指在别人早已买进后才醒悟，也跟着买进，将股价抬高，虽然后续也许能盈利，但会让之前买入的人获利更大。

# 放量、缩量

放量：指成交量比之前有明显放大。例如，某只股票昨天全天的成交量是1亿股，今天忽然变成2亿股，这就是放量。 缩量：与放量相反，是指成交量比之前有明显的减少。

# 天量、天价

天量：表示某只股票或整个市场当天出现的巨大的交易量，远远超过平时的成交量。 天价：指股票的价格达到一个较高的水平，一般表示股票已经处于上升的末期，离未来的下跌为时不远。

# 热门股、冷门股、白马股、龙头股

热门股：是交易量大、流动性强、价格变动幅度大的股票。 冷门股：是交易量小，流动性差甚至没有交易，价格变动小的股票。 白马股：一般是指其有关的信息已经公开，业绩优良，具有成长性，低风险的特点，因而具备较高的投资价值，往往为投资者所看好。 黑马股：是指脱离过去的价位而在短期内大幅上涨的股票。黑马股可遇而不可求，如果被大家都看好的股票就很难成为黑马股了。 龙头股：在股票市场的炒作中，对同行业板块的其他股票具有影响力和号召力的股票，它的涨跌往往对其他同行业板块股票的涨跌起引导和示范作用。

# 蓝筹股、绩优股、红筹股

蓝筹股：“蓝筹”一词源于西方赌场。在西方赌场中，有三种颜色的筹码，其中蓝色筹码最为值钱，红色筹码次之，白色筹码最差。投资者把这些行话套用到股票上，“蓝筹股”指在某一行业中处于重要支配地位，业绩优良，实力雄厚的大公司股票，在一些地方也将蓝筹股视为"绩优股"。蓝筹股一般会有稳定的分红，但公司已经规模很大，并不会有高速成长，所以很难有非常高的收益。

红筹股：这一概念诞生于90年代初期的香港股票市场。中华人民共和国在国际上有时被称为红色中国，相应地，香港和国际投资者把在境外注册、在香港上市的那些带有中国大陆概念的股票称为红筹股。

# 大盘股、中小盘股

大盘股：指市值较大的大型公司股票。 中小盘股：与大盘股对应，指市值较小的中小型公司股票。

# 现手

现手：指某只股票最新成交的一笔的股票数量。

# 盘口

盘口，在股市中一只股票的买卖信息，比如最新的买方报价和卖方报价、最新成交价等。

# 量比

量比：是指衡量相对成交量的指标。它是开市后每分钟的平均成交量与过去5个交易日每分钟平均成交量之比。量比是用来判断当日成交活跃度的。

# 委比

委比：指用以衡量一段时间内买卖盘相对强度的指。其计算公式为： 委比＝〖(委买手数-委卖手数)÷（委买手数＋委卖手数)〗×100%

# 内盘、外盘

内盘：指股票在买入价成交的数量，如果内盘大，说明卖方愿意以买方的报价成交的意愿强。 外盘：指股票在买入价成交的数量，如果内盘大，说明买方愿意以卖方的报价成交的意愿强。 外盘和内盘的比，可以看出市场买方和卖方的意愿对比。

# 均价

均价：是某一时刻买入或卖出股票的平均价格。

# 含权

含权：凡是有股利的股票，尚未除权，即为含权。

# 回购

回购：是指上市公司利用现金等方式，从股票市场购回本公司发行在外的一定数额股票的行为。

# 对冲

（Hedge）

对冲：指持有两种相关性为负的资产，比如A和B，A如果跌，B就会涨；B如果跌，A就会涨，所以持有两种资产总有一种资产是会涨的，不会两种资产都亏钱。

# 锁单

锁单：指持有一种资产的情况下，立刻买入或卖出一种资产，形成对冲，来锁定自己当前账户的净值。锁单一般要求两种资产的相关性非常高，比如买入标普500 ETF基金的同时，卖出标普500期货合约。

锁单的目的一般是比如临时有事、吃饭上厕所，不得不离开，又或者仓位要隔夜，第2天市场走势会有巨大的不确定性。由于不想在自己离开期间、或者第2天开盘时，市场突发不利的走势而自己损失惨重，所以使用用锁单来保护。

# 黑天鹅

（Black Swan）

欧洲人一直到十七世纪澳大利亚被发现为止，都觉得世界上的天鹅都是雪般无瑕的白天鹅。但当1697年，荷兰探险家威廉佛拉明（Willem de Vlamingh）领导的探险队，在探索澳大利亚西部珀斯“天鹅河”（Swan Lake）的过程中首次发现黑天鹅时，欧洲人吓了一大跳。因此，人们也开始把非同寻常且难以预测的事件叫做“黑天鹅事件”。

但是在股市上，黑天鹅事件并不罕见，而通常都是对股市不利的消息，导致股市大跌。久而久之，黑天鹅在金融圈中也表示未知风险，在交易时，应当重视黑天鹅事件，虽然黑天鹅无法预测，但是交易时应该考虑“如果突然发生黑天鹅，我持有的资产下跌，我该怎么操作”。

# 灰犀牛

（Grey Rhino）

灰犀牛”一词则源自2013年1月，由美国学者Michele Wucker所提出，2016年他出版《灰犀牛：如何认清我们忽视的明显危险并采取行动》（The Gray Rhino: How to Recognize and Act on the Obvious Dangers We Ignore）一书后，才被广为流传。

“灰犀牛”是指一些已经表现得很明显、因素、威胁和警告，如果不重视，迟早会无法控制而引起严重后果。这就类似灰犀牛，这是一种生长在非洲大草原，身躯庞大，给人一种行动迟缓、安全无害的错觉，当他们开始起跑的时候可能不是很快，但是加速度非常快，一旦灰犀牛狂奔起来以后，力量和速度都非常惊人，这种无法阻止的冲击力如果撞在某个人或者某辆车上，都是巨大的灾难。

# 融涨和害怕错过

融涨（melt up）和害怕错过（FOMO）

所谓融涨，英文名melt-up，是指投资者由于不想错过股市上涨的机会，而蜂拥入市场买买买，导致资产价格不断上涨的情况。这种购买不是因为经济基本面有所改善，而是一种害怕错过的焦虑性心理---FOMO导致的。

FOMO，英文全称是 Fear of Missing Out，意为“害怕错过”，最初发现这一现象的是营销专家 Dan Herman，他在 2000 年发表的一篇名为《引入短期品牌：新的消费者现实下的新品牌推广工具》的文章中提到“错失某事的恐惧”，而“FOMO”这个词语，则是由作者帕特里克J. 麦金尼斯创造，并于 2004 年在《麦金尼斯的两个 FO：哈佛商学院的社会理论》一文中阐述了其具体意义，2013 年，“FOMO”一词被加入牛津词典，指代的是人们害怕错过社交媒体上发生的事情，从而产生焦虑心理。

现如今，FOMO 不单单指代社交媒体触发的焦虑，同时也表现了人们在面对某个新事物时产生的恐慌 。延伸到投资领域来说，如果一个人受到 FOMO 的影响，则表现为做交易决策时患得患失，缺乏清晰的判断力，如此一来便倾向于跟风操作。

产生FOMO的情绪，通常是由“当初不敢买，如今买不起”的经历造成的，比如一些人曾经在中国房价暴涨、亚马逊股票暴涨、比特币暴涨之前有过“要不也买吧”的想法，在那个时候他们是有本金上车的，但一犹豫就错过了，事后非常后悔，尤其是看着曾经身价不如自己、能力不如自己的人快速致富后，心理就更加焦虑了，此后当他们发现某个他们认为的好资产时，他们会毫不犹豫的买入，而忽视宏观大环境和资产价格是否高昂，事实上，他们甚至没有花时间深入研究过这些问题，而只是看到身边有人在买这个赚钱了，再加上媒体和别有用心的推手不断洗脑“再不买就永远买不起了”，于是彻底丧失了理智做出“追涨”的行为。

在经济并不具备稳步上涨的条件时，却有可能出现大量的FOMO使得市场融涨，最终达到一定程度产生资产泡沫化。

值得一提的是，当泡沫破灭时，那些因为FOMO而买入资产的人，通常是最先割肉，或者说被收割的人，因为他们并没有对自己买入的资产有深入的研究和长期坚定的信仰。

# 常用买卖订单类型

另外，对于交易所来说，只有市价单和限价单两种基本订单。其他的订单类型，是券商的软件系统提供的，最终券商的系统会帮助监控价格，然后达到条件后就转为市价单和限价单，发给交易所。

# 市价单

市价单，以市场价格买进或卖出的委托单：

  • 买入市价单（buy market），不管当前的卖方报价是多少，立刻买入。
  • 卖出市价单（sell market），不管当前的买方是多少，立刻卖出。

市价单没法准确控制成交价，这有时候很危险，比如，你看到卖方报价当前是100，此时你下市价单买入，但实际情况有可能是，当你的买入指令发到交易所时，那个100的卖方报价，已经被成交或者撤销了，而最新的最低卖方报价是110，那么你会110的价格买入，比你逾期的成本高了10%！

# 限价单

  • 买入限价单（buy limit ）：以指定的价格A在交易所挂单，当市场卖方报价，达到该价格或以下时，订单即成交，成交价，一定是你设定的价格A。买入限价单对于做空时的回补操作也有效。
  • 卖出限价单（sell limit ）：以指定的价格A挂单，当市场买方报价，达到该价格或以上，则执行卖出操作。所以卖出限价的成交价，一定是你设定的价格A。

限价单是对整个市场是可见的，参考股票基础知识的盘口概念，这个一定程度上暴露了自己的意图，如果想隐藏，可以考虑“触及市价单或触及限价单”

另外，如果这个证券成交量不够，不一定能在限价A上成交成功，或者会有部分成交的情况。后续凡是带有限价特点的订单，都是如此。

# 止损单

止损，表示预期价格上涨或下跌，但实际情况是价格向相反的方向运动，于是做出减少损失的行为。

  • 买入止损单（buy stop）：预期价格会下跌，以指定的价格A挂单，当市场买方报价，达到该价格或以上，则买入。这种订单主要用于两种情况：
    • 做空止损：预期价格会下跌，但是预期错误达到止损价，则平掉做空仓位。
    • 追涨：一旦价格上涨突破止损价，认为会有一波上涨趋势，立刻买入。
  • 卖出止损单（sell stop）：预期价格会上涨，以指定的价格A挂单，当市场卖方报价，达到该价格或以下，则卖出。这种订单主要用于两种情况：
    • 做多止损：预期价格会上涨，但是预期错误达到止损价，则平掉做多仓位。
    • 做空：一旦价格下跌突破止损价，认为会有一波下跌趋势，立刻做空。

# 触及市价单

触及市价单（market if touched），这种订单和市价单很类似，不过需要设定一个触及格：

  • 买入触及市价单：当市场价格下跌到触及价时，立刻以市价买入。注意，这里是以市价买入，而不是限价单里设置的限价A。
  • 卖出触及市价单：当市场价格上涨到触及价时，立刻以市价卖出。注意，这里是以市价买入，而不是限价单里设置的限价A。

触及市价单是保存在券商的系统中，直到价格触发到了触及价，才会发送给交易所，所以隐藏了自己的报价，不过它同样面临着市价单的风险，没法控制成交价。

# 触及限价单

触及限价单（limit if touched），这种订单在触及市价单的基础上，加入了限价操作，你需要设定一个限价A，在市场价格触发触及价后，会发送限价单给交易所，限价为你设定的A。

以买入触及限价单为例，你可以设定触及价格为100：

  • 可以设定限价如101，当市场价格跌到100以后，你会发送101的限价订单给交易所，这有可能能迅速成交而避免了你的，同时又避免了像市价单一样冒着110的成交价格风险无脑买入。
  • 可以设定限价如99，当市场价格跌到100以后，你会发送99的限价订单给交易所。这种方式相当于。在价格到达触及价之前，你都隐藏了自己的限价单，避免被所有人看到。

# 止损限价单

  • 买入止损限价（buy stop limit）：设置一个限价和一个止损价，限价必须大于止损价，比如限价为20，止损为19，则价格上涨突破19以后，则以限价20的价格挂单买进（<=20就买进）。这种订单和买入止损单的作用类似，之所以设置额外的限价，是避免股价从10突然涨到30后，你不想以市价30买进。
  • 卖出止损限价（sell stop limit）：设置一个限价和一个止损价，限价必须小于止损价，比如限价为19，止损为20，则价格下跌跌破20以后，则以限价19的价格挂单卖出（<=19就卖出）。这种订单和卖出止损单的作用类似，之所以设置额外的限价，是避免股价从30突然跌倒10后，你不想以市价10卖出。

# 移动停损单

买入移动停损单（but trailing stop)，假设设置初始止损价是260：

  • 输入订单，设定尾随金额2，则初始停损价262。
  • 股价下跌到255。
  • 止损价自动调整到257。
  • 股价上涨到256，移动止损价不变还是257。
  • 股价下跌到250。
  • 止损价自动调整到252。
  • 股价上涨到253，触发设定的停损价，以市价买进，止盈。

卖出移动停损单（sell trailing stop)，假设设置初始止价格是260：

  • 输入订单，设定尾随金额2，则初始停损价262。
  • 股价上涨到268。
  • 止损价自动调整到266。
  • 股价下跌到267，移动止损价不变还是266。
  • 股价上涨到275。
  • 止损价自动调整到273。
  • 股价下跌到273，触发设定的停损单，以市价卖出，止盈。

如上，很明显，移动停损单是一种带有止盈性质的订单，当价格向你预期的方向移动时，会动态的将止损价（其实是止盈价）向预期方向调整，以备及时止盈。

# 移动停损限价单

移动止损限价单（trailing stop limit），和移动停损单是类似的，不过除了设定止损价，还要设定一个限价抵消（limit offset），当最后触发停损时，不是以市价买进或卖出，而是以限价买入或卖出。

  • 买入移动停损限价单；限价=停损价+限价抵销，以限价买入
  • 卖出移动停损限价单；限价=停损价-限价抵销，以限价卖出

收盘限价单（Limit On Close） 收盘限价定单（limit on close），和限价单类似，不过是在收盘时，对收盘价判断：

  • 买入收盘限价单：收盘价小于或等于限价A时，以收盘价格买入。
  • 卖出收盘限价单：收盘价大于或等于限价A时，以收盘价格卖出。

这里的收盘价格肯定要比A，所以券商的系统会长尝试用收盘价去成交。

# 开盘限价定单

开盘限价定单（limit on open），和收盘限价单类似，不过是以开盘价来判断。
