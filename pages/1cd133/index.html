年大之如何逃顶

# 简介

抄底与逃顶，是交易的两大难题。相对而言，抄底更容易，逃顶更难。但不管是抄底，还是逃顶，都是有一些技巧的。

不管是大盘的顶部，还是板块与个股的顶部，也都是有迹可循的。

今天，主要讲述个股与板块的逃顶，大盘的逃顶，将在**《如何分析大盘》**的科目中讲述。

哥曾经吹过几次牛皮：

哥看多的胜率，只有60-70%；但哥看空的胜率，几乎是百分之百，被哥看空的品种与标的，几天或几周后，无一例外的都出现了暴跌。

不管是1月中旬看空的海狗，还是春节前看空的白酒医药新能源汽车与光伏，或是白酒的大C浪杀跌，以及最近新能源汽车顶部的风险提示，基本都得到了市场的验证。

当然了，有些杠精会说，曾经被哥看空品种与标的，后来不少又新高了，纯属没有意义的抬杠。一波40%左右的暴跌，能躲过难道不香吗？如果你那时满仓满融，一波40%左右的暴跌，你扛得住吗？典型的“死鸭子嘴硬”。

今天的科目，不能让你多赚钱，但可以让你少赔钱。

言归正传。

# 个股的顶部识别

底部看逻辑，顶部看K线。当股价炒上天际后，莫要相信鬼故事，而是要跟着趋势走。

顶部的识别与逃顶，除了突发事件之外，绝大多数情况下，都需要通过技术分析来判断。

以海狗为例，来讲述如何逃顶。

# 1月中旬逃顶

大牛股见顶，很容易出现在两个位置：一个是5倍股附近，一个是10倍股附近。

海狗的第一个顶部，即左侧顶，就是出现在5倍附近。

1月14日，海狗放量跌停，就是一个危险信号，初步可以判断顶部出现了。因为，对于任何大牛股来说，涨了5倍10倍后，高位放量跌停，都需要格外关注，见顶概率很大。

1月19再次放量跌停时，顶部得到完全确认。如果等到20天线跌破后，你才反应过来见顶了，那时已经太晚了，快30%都没了，要是用了杠杆，可能50%都没了。

在实际操作时，我们并不需要等到1月19再次跌停，才开始减仓或卖出。1月14日第一个跌停出现，或即将跌停时，就需要大幅减仓了，至少要把杠杆去掉。

如果还想再看看，那么可以结合周线来操作，当海狗的第一个跌停板出现后，周线已经是明显的顶部形态了，这个时候，大幅减仓是没有问题的。

卖对了，你可以躲过一波接近腰斩的暴跌；万一卖错了，后面等形态修复后，再买回来也没啥，“树是死的，人是活的”，要灵活变通。

# 7月中旬逃顶

海狗的第二个顶部，即右侧顶，出现在10倍附近。

这次虽然没有出现高位放量跌停，但是，7月9日放量大跌，而且60分钟线带了缺口，周线出现了顶部迹象，这是一个需要减仓的信号，至少需要去掉杠杆。

7月15再次放量大跌，日线带了缺口，同时击穿了20日均线，顶部信号已经很明显了，这个时候，大幅减仓或者清仓，都没有任何问题。

因为，海狗已经涨了10倍了，再去继续博弈，风险与收益明显不成比例，何况它还是强周期股，而不是成长股，20倍的大牛股或英科医疗那种超级大牛股，都是可遇而不可求，都是极小概率事件。

# 板块的顶部识别【要点】

板块的逃顶，主要分为两种：

  • 一种是A浪逃顶，也就是5浪主升浪后逃顶；
  • 一种是大C浪逃顶，也就是B浪反弹后逃顶。

至于如何识别主升浪与B浪反弹，自己去看6月6日晚的文章。

热门板块的见顶，很容易出现在两个时点：

  • 一个是每年的春节前后，即形成5浪顶或左侧顶；
  • 一个是每年的7月左右，即形成B浪顶或右侧顶。

因为，热门板块的容量都很大，市值都是数千亿起，甚至万亿以上，里面进的资金太多，一个顶部根本无法完成出货，所以，都会有两个顶部。

每一个大的热门板块，特别是主流品种，基本都会遵循“5浪上升+3浪下跌”的结构，中间会出现1浪、3浪、5浪、B浪等4个上升浪，2浪、4浪、A浪、C浪等4个下跌浪。

其中，2浪与4浪，是两个调整浪，不是主跌浪，我们不需要逃顶，也没必要逃顶，因为很容易卖飞在主升前，而且，它们的调整幅度也不会太大，龙头一般20%左右打住，大多标的25-30%打住。

但是，A浪与C浪，是两个主跌浪，是需要逃顶的，否则损失会非常大，如果高位满仓满融的，即使没被消灭掉，也会只剩下一口气。

因为，这两个主跌浪的跌幅非常大，都是40%的跌幅起步，最强的龙头大多时候也会跌35%左右，大多标的会跌45%左右，杂毛股经常会被腰斩。

对于板块的逃顶来说，我们需要重点去关注几只股票：板块龙头、人气龙头、核心标的。

当一个板块要阶段性见顶时，通常人气龙头会率先高位跌停，多只核心标的跌停或暴跌，板块龙头接近跌停或大跌。

# A浪逃顶

A浪的逃顶，通常都是在每年的春节前后，大盘走强配合年报，主力完成第一次出货。

以去年的芯片为例，来讲述如何A浪逃顶。

去年芯片股的5浪主升浪顶部，出现在去年的2月26日，板块集体见顶。

板块龙头韦尔股份：2月26日，放量大跌，接近跌停，周线也出现了顶部迹象。

人气龙头晶方科技：2月26日，放量跌停，周线也是明显的顶部迹象。

与此同时，核心标的卓胜微、闻泰科技、兆易创新、圣邦股份等，都是类似，不是放量跌停，就是接近跌停。

而且，板块的核心标的，基本都实现了10倍左右的涨幅，大多标的都涨了5倍左右，估值基本都在100倍以上。

这个时候，当板块集体暴跌时，基本就可以确定顶部了。此时，不要再相信任何故事，要跑了再说，否则一波40%左右没有了。

2020年7月中旬时，芯片股在完成了B浪反弹后，又形成了第二个大顶，即右侧顶，主力完成了第二次出货后，芯片横盘震荡了一年左右，大多标的都跌破了年线。

直到今年二季度，芯片才再次盘出底部，但即使到现在，芯片板块也还没再次走出趋势性大行情，而只是结构性行情。

# 大C浪逃顶

C浪的逃顶，通常都是在每年的7月前后，大盘走强配合中报，主力完成第二次出货。

如果说，A浪逃顶没逃掉，还有得救，不会死透，因为主力无法一次完成出货，通常都会有个B浪大反弹，会再给一次逃命的机会。

但是，大C浪逃顶，如果没逃掉，除非手里拿得是最牛逼的那几只标的，否则会死的很惨，**因为B浪大反弹之后，主力基本完成了尾货的出清，未来一年半载左右，**绝大多数标的都会跌破年线，沦落为熊股，C浪杀完之后，板块估值会大幅下滑，除了极少数标的还能继续新高外，绝大多数标的，一两年都难以新高。

去年三季度之后的芯片5G消费电子如此，今年三季度后的白酒医药新能源汽车光伏也会如此。

股市从来就没有新鲜事，历史总是在不断重复，因为人性总是在不断重复。

那些认为“这次不一样”的人，最后都成为了高位的接盘侠。

以今年的新能源汽车为例，来讲述如何C浪逃顶。

板块头宁德时代：7月16日，高位放量大跌，离10倍近在咫尺。

人气龙头石大胜华：7月14日放量跌停，提前见顶，7月16日，再次放量大跌，最大涨幅接近20倍，竞争对手华鲁恒升要来了，对它的逻辑会有冲击。

核心标的比亚迪：整车龙头，也是龙二，这波没创新高，走势算比较弱的一个了，因为今年业绩不够出彩估值太贵所致，先于板块见顶，最大涨幅接近7倍。

核心标的恩捷股份：隔膜龙头，7月16日，放量跌停，最大涨幅接近13倍，恒力石化也要搞隔膜了，对它的逻辑会有冲击。

核心标的天赐材料：电解液龙头，7月14日，放量大跌，接近跌停，7月16日，再次放量大跌，最大涨幅接近19倍，多氟多扔出来了个重磅炸弹，对它的逻辑冲击不小。

新能源汽车的顶部，基本得到了确认，即使后面还有反复，基本也是顶部区域了。

其实，7月14日，部分核心标的跌停或暴跌时，就是一个危险信号了，在人气龙头石大胜华跌停时也提示过了风险。这个时候，你即使不大幅减仓，也需要去掉杠杆，去弱留强。

7月16日，核心标的集体大跌，龙头宁德也大跌时，已经不需要再犹豫了，先跑了再说，否则，再过几天40%左右可能就没了。

当然了，对于新能源汽车，也不用一棒子打死，它与芯片一样，都是最好的赛道，这轮暴跌之后，依然还有一些大机会。

但是，大的趋势性行情，可能短期看不到了，将从趋势性行情，向结构性行情演化，业绩增速高的公司，还会继续新高，业绩垃圾的杂毛，则将沦落为熊股。

同时，又会走出一批新的小姐姐，成为新的5倍10倍股。这轮杀跌之后，重点关注PVDF的机会，大概率是下一个6F，而且，逻辑比6F更硬。
